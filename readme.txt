# HTML5 / CSS / Javascript

Create a website for a local bookstore in Singapore. 

In  website, the following have been implemented:
= Layout Tables
= Colspan and rowspan in tables
= Links and URL relative referencing
= Figure and caption
= 1 external CSS file for all webpages
= Measurements need to be used in the CSS
= Overflow properties
= An appropriate image should be used for at least one of the buttons 
= HTML5 date controls 
= Positioning 
= Floating elements 
= Hovers used to make thumbnails
= Multiple HTML files
= Only 1 external JavaScript file
= JavaScript constructor, properties and methods
= Variables, comparison operators, logical operators
= Functions and alerts
= Listener approach
